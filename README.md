# CAPM Exam Navigator

Free CAPM study web app with lessons, practice drills, mock exams, and local saved progress.

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run check:questions
npm run build
```

## Deploy

This project is configured for GitHub Pages at:

```text
https://naoufalatm.github.io/capm-exam-navigator/
```

Pushing to `main` runs the GitHub Actions workflow in `.github/workflows/deploy.yml`.
